SELF ASSESSMENT EMAIL HANDLER

POST: /aws/build-email

Post data to generate an email build based on a template and language

{ "templateId": "reset-password", "data": { "name" : "Sam Jones", "email" : "sam@gmail.com" }, "language": "en-GB" }


GET: /aws/user-email

Route to preview the generated html email template with passed data


GET: /aws/user-email-text

Route to preview the generated txt email template with passed data


GET: /templates

Route to preview a folder directory of the html and text emails with populated data


GET: /templates-hbs

Route to preview a folder directory of the html and text emails with handlebars


GET: /templates-user

Route to preview a folder directory of the html and text emails generated with user data


GET: /cms

Route to preview local cms files for testing
